# Color Generator using jaon server

A color generator using json server you need to install npm and run dev usin 'npm run dev'.

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm run dev
```

### Build
Compile and build

```sh
$ npm run build
```
